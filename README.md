# Portfolio Małgorzaty Klimczuk — wersja designerska

Nowoczesne, responsywne portfolio typu one-page do publikacji na GitHub Pages.

## Publikacja
1. Utwórz repozytorium na GitHubie.
2. Wgraj wszystkie pliki z folderu.
3. Wejdź w Settings → Pages.
4. Wybierz Deploy from a branch → main → /root.
5. Zapisz.

## Przed publikacją
- Podmień linki `href="#"` na prawdziwe adresy projektów.
- Dodaj prawdziwe screeny, jeśli chcesz zastąpić mockupy CSS.
- Opcjonalnie dodaj link do LinkedIn i plik CV.
